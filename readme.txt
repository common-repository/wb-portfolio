== Description ==

WB Portfolio is a custom post type based  Responsive Filterable Portfolio showing plugin. WB Portfolio plugin works using shortcodes [wb-portfolio].
Also can be add in themes PHP template  using  <?php echo do_shortcode('[wb-portfolio]'); ?>

You can display Portfolio list anywhere in your site using this shortcode . It is responsive and easy to use.

<strong>Shortcode</strong>

<pre>[wb-portfolio]</pre>
